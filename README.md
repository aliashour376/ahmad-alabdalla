# Ahmad Al Abdalla

Lebanon-first website for Ahmad Al Abdalla.

## Commands

```powershell
npm install
npm run dev
npm run build
```

The site uses real public brand/menu content gathered from the incumbent website, Instagram metadata, and the deployed Lebanon menu app.

## GitHub Pages

Pushes to `main` build and deploy automatically with GitHub Actions.

Published URL: `https://aliashour376.github.io/ahmad-alabdalla/`

### Rollback

Revert the faulty commit on `main` and push the revert. The Pages workflow will publish the restored build.
